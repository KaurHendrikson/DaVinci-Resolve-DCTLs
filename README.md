## Installation

Copy .dctl files over to your LUT folder and restart DaVinci Resolve

### Windows

`C:\ProgramData\Blackmagic Design\DaVinci Resolve\Support\LUT`

### Mac

`/Library/Application Support/Blackmagic Design/DaVinci Resolve/LUT`

### Linux

`/home/resolve/LUT`
or
`/opt/resolve/LUT`
